TRACKING COVID-19 ACROSS THE US



Definitions and Explanations

Total, Daily and 7 Day Average:
  • The total cases and deaths are pulled directly from the John's Hopkin's github repository.
  • Daily cases are found by calculating the change in total cases from the previous day. Daily cases and deaths are sometimes negative, this happens when the a state or county changes the total cases or deaths after the original reporting.

  • The 7 day average is calculated by taking the previous week from the current date and averaging the daily cases or deaths.

States and Counties:
  • The Johns Hopkins Github repositiory give data for each county and for "unassigned" and "out of state" within each state. The numbers we use for states are the summation of all three. However, the unassigned and out of state cases do not appear in the county data.
Order Dates:
  • The order dates are from FINRA. These dates are statewide "Shelter-in-Place" or "Stay-at-Home" orders; however, certain counties might have had stricter quarantine periods.
Machine Learning Projections

  • Cases Projections
    • A TensorFlow nueral net was used with a ReLU activate. The variables used were Population Density, Black population share, Hispanic population share, Median age, Current case total, 7 Day average case totals, and the 7 day average of close proximity cases.
    • All data is pulled from the Johns Hopkins Github Repository, or the Python Census API using the 2019 census eastimates
    • Close proximity cases are calculated by summing every county within a 5 degree radius from the county in question. This is calculated using the latitude and longitude for each county on the Johns Hopkins Github.
  • Death Projections
    • A TensorFlow nueral net was used with a ReLU activate. The variables used were Population Density, Black population share, Hispanic population share, Medain household income, Median age, Current case total, 7 Day average case totals, and the 7 day average of close proximity cases.
    • All data is pulled from the Johns Hopkins Github Repository, or the Python Census API using the 2019 census eastimates
